NaadanRoute

A sophisticated travel ecosystem engineered at the intersection of web architecture and intelligent data systems. Driven by Kera AI, the platform leverages Python-driven data models to provide a personalized, contextual experience for the modern traveler.

Technical Stack

  • Backend: Django, Python (MVC Architecture)
  • AI Engine: Kera (RAG-based Architecture)
  • Database: MySQL (Production) / SQLite3 (Dev)
  • Frontend: CSS, Bootstrap, JS, jQuery
  • Tools: Git/GitHub

Key Features

  • Kera AI: GenAI-driven Conversational Assistant
  • Smart Discovery: RAG-based Package Recommendations
  • Secure Payments: Integrated Razorpay Gateway
  • User Auth: Profile Dashboard & Session Management
  • Data Systems: CRUD-based Wishlist & Blog Architecture
  • UI/UX: Fully Responsive Cross-Browser Design
